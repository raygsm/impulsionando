# Templates Impulsionito (in-app IA)

Impulsionito = assistente in-app. Mensagens curtas (≤ 240 caracteres),
com CTA único. Renderizadas em bubble e/ou toast.

## Onboarding

**imp.onboarding-d0**
```
Oi {{customer_name}}, sou o Impulsionito 👋 Vamos configurar seu
{{module_name}} em 3 passos? → {{cta_url}}
```

**imp.modulo-nao-configurado**
```
Vi que o {{module_name}} está esperando você. Levo você em 2 minutos:
{{cta_url}}
```

## Uso

**imp.sugestao-recurso**
```
Você usa muito {{module_name}}. Descobri um atalho que economiza tempo:
{{cta_url}}
```

**imp.impulsionito-proativo**
```
Notei um padrão nos seus dados — posso preparar um resumo para você?
{{cta_url}}
```
